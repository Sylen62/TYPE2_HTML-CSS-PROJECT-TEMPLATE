Position

Position, tai HTML elemento pozicionavimas kitų elementų atžvilgiu. Standartinis pozicionavimas vadinamas static.

Pagrindiniai position tipai

  • static
  • relative
  • absolute
  • fixed
  • sticky

Static

Static pozicionavimas veikia rašymo tvarkos principu: iš kairės į dešinę, o jeigu elementai netelpa į tėvinio elemento plotį, tuomet persikelia į sekančią eilutę.

Relative

Relative - elemento pozicionavimas, pagal standartinę static elemento poziciją. Tam naudojamos CSS savybės:

  • top
  • right
  • bottom
  • left

Absolute

Absolute - pozicionavimo tipas, kuomet yra ieškomas pirmas tėvinės hierarchijos elementas, kurio position nėra static. Radus tokį elementą, atsispiriama nuo jo kontūrų naudojant savybes top, right, bottom, left. Jeigu tėvinės hierarchijos šakoje nėra randamas elementas su ne static position tipu, elementas atsispirs nuo vieport kontūrų, matomų puslapio viršuje. Nustačius position: absolute, kiti elementai jo nebemato ir yra išdėstomi, tarsi jo nebūtų.

Fixed

position: fixed elementai visada išlieka toje pačioje viewport vietoje. Atsispyrimui nuo vieport naudojame savybes top, right, bottom ir left.

Šio puslapio navigacinė juosta (navbar) pririšama prie matomo lango (viewport) naudojant position: fixed;

Sticky

Sticky - pozicionavimo tipas, kuomet ieškomas elementas tėvinėje hierarchijoje, kuriame yra įgalintas scroll'inimas. Tuomet nuo to elemento kontūrų, yra atsispiriama naudojant CSS savybes top, right, bottom, left.
